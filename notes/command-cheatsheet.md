# Command Cheatsheet
 
| Command | Fungsi |
|---|---|
| pwd | Melihat isi folder |
| ls | Melihat isi folder |
| ls -la | Melihat semua file termasuk |
| cd | Pindah Folder |
| mkdir | Membuat folder baru |
| touch | Membuat file kosong |
| cat | Membaca isi file |
| cp | Menyalin file atau folder |
| mv | Memindahkan atau Rename File |
| rm | Menghapus file |
